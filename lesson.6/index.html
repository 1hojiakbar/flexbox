Flexbox Asoslari

Flexbox: - float yoki position xossalari yordamisiz veb sahifada moslashuvchan ( flexible ) layout yaratishda yordam beradi. Boshqacha qilib aytganda, veb sahifadagi elementlarni joylshtrish uchun xizmat qiladi. Flexbox 2 asosiy elementlardan tashkil topgan: Flex container Flex item

Flex container - display xossasi uchun flex qiymatiga ega bo'lgan elementga aytiladi.

Flex Model: - asoslari quydagilardan tashkil topgan: flex-container flex-item asosiy o'q(main axis) va kesuvchi o'q(cross axis)
